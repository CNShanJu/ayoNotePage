# 其他样式

## 关于input标签选中边框

```css
input{
    border:none；
    
    /* 可以去掉元素选中默认出现的边框 */    
    outline:none;
}
```

## 显示省略号

```css
p{
    width：100px；
    border：1px solid black;
    
    /* 超出显示省略号 */
    text-overflow:ellipsis;
    
    /* 不换行 */
    white-space:nowrap;
    
    /* 超出隐藏 */
    overflow:hidden；
}



```

## box-sizing属性

```css
div{
	width:200px;
	height:200px;
    background:red;
    /* 把设置的边框、内边距全都计算到设置的宽高属性中 */
    box-sizing:border-box;
    /* 默认属性，设置的宽高只计算 里面的内容 占据的宽高 */
    /* box-sizing:content-box; */
    
    
    
    padding:10px;
    border: 50px solid black;
}


```
